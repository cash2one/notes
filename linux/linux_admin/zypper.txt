列出配置的软件源，显示详情（优先级、网址等等）：
zypper repos -d

要查询特定包的所有安装源，请使用 search。它用于包名称或（可选）包摘要和描述。搜索项中
允许使用通配符 * 和 ?。默认情况下搜索不区分大小写。
zypper search vsftpd 

安装软件
zypper install vsftpd(软件名) 

卸载某个软件包
zypper remove package_name

升级某个软件包
zypper update package_name
