下载一个代码库,修改,上传
1 git clone https://github.com/xuyuanxin/notes    下载代码
2 git add .                                       添加文件到暂存区
3 git commit -a -m "message"                      提交到本地版本库
4 git push origin master                          提交到github

---->回退修改
git checkout -- filename  回退filename的修改(进入到文件所在的目录)


----> 删除文件
1 git rm filename
2 git commit filename

----> 文件重命名
1 git mv <旧文件名> <新文件名> 
2 commit即可
